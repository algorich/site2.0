Trabalhe conosco

Por que fazer parte da Algorich?

Porque aqui você...

  • Irá adquirir experiência profissional com as metodologias e ferramentas de desenvolvimento mais recentes do mercado
  • Terá um ambiente de trabalho descontraído e produtivo com foco no aprendizado
  • Fará parte do crescimento da empresa de desenvolvimento web mais promissora da região!

Estamos em busca de novos integrantes para nosso time!

A Algorich está crescendo e chegou o momento de aumentar nosso time!

Estamos com duas vagas de estágio em desenvolvimento de software com início em XX/04/2013. Se você tem disposição para aprender, aqui você vai trabalhar com:

  • Desenvolvimento web com Ruby on Rails
  • Desenvolvimento guiado por testes
  • Desenvolvimento de aplicativos para dispositivos móveis com PhoneGap
  • Ambiente de desenvolvimento Linux
  • Controle de versão com Git

Aqui você vai encontrar um ambiente descontraído e confortável, excelente para trabalho em equipe, além de um escritório equipado com tudo para fazer um bom lanche! =)

Processo seletivo

O processo seletivo seguirá as seguintes etapas:

  1. Preencher o formulário de inscrição que se encontra logo abaixo e estará aberto até o dia 17/05/2013.
  2. No memso dia vamos enviar por email um teste: Criar uma aplicação (vamos dizer qual no email) usando Ruby on Rails que deverá estar rodando no Heroku até às 23:59 horas do dia 24/05/2013. É importante lembrar que você deve utilizar sua criatividade para aplicar seus conhecimentos nos itens listados como requisitos e conhecimentos desejáveis.
  3. Vamos bater um papo com cada um dos selecionados na etapa 2
  4. Até o dia 29/04/2013 vamos informar os dois selecionados

Observação: se você não adora a área de desenvolvimento e/ou apenas quer fazer este estágio porque é obrigatório para concluir o seu curso, por procure outra empresa. É essencial que os candidatos sejam apaixonados por desenvolvimento.

Detalhes da vaga

  • 20h semanais
  • Bolsa de R$ 500,00 + auxilio transporte
  • Escritório no centro da cidade, em frente à estação central de ônibus (veja no google maps)

Requisitos

  • Estudantes de cursos de nível superior na área de informática
  • Conhecimento básico de Ruby, Orientação a Objetos, Ruby on Rails, HTML e SQL

Conhecimentos desejáveis

  • Escrever e, principalmente, ler em inglês
  • praticar TDD (bônus para Rspec e Capybara)
  • Linux ou OS X
  • Git
  • JavaScript
  • CSS

Diferenciais

  • Vontade de aprender
  • Pró-atividade
  • Ser atuante na comunidade e colaborar com projetos de software livre
  • Criatividade
